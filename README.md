# 🌐 Personal Portfolio Website

## 🚀 About the Project

This is my personal portfolio website built to showcase my skills, projects, and experience as a **Full Stack MERN Developer**.
The website highlights real-world projects, technical expertise, and provides an easy way to get in touch with me.

## 🔗 Live Demo

👉 https://portfolio-9ja.pages.dev/

## ✨ Features

* 📱 Fully responsive (mobile, tablet, desktop)
* 🎨 Clean and modern UI design
* ⚡ Smooth animations and transitions
* 📂 Projects showcase with live/demo links
* 🧠 Skills section with experience-based levels
* 📄 Resume download option
* 📬 Contact form (Formspree integration)
* 🌙 Light/Dark mode toggle
